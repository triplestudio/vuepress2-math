问题 1: 二次函数求最值

题目描述

已知二次函数 f(x)=ax2+bx+cf(x) = ax^2 + bx + cf(x)=ax2+bx+c，其中 a>0a > 0a>0。求函数 f(x)f(x)f(x) 在区间 [m,n][m, n][m,n] 上的最大值和最小值。

解题思路

对于二次函数 f(x)=ax2+bx+cf(x) = ax^2 + bx + cf(x)=ax2+bx+c，其对称轴为：

x=−b2ax = -\frac{b}{2a} x=−2ab​

  1. 如果对称轴在区间 [m,n][m, n][m,n] 内，则最小值在对称轴处取得，最大值在区间端点处取得
  2. 如果对称轴在区间左侧，则函数在区间上单调递增
  3. 如果对称轴在区间右侧，则函数在区间上单调递减

解答过程

设对称轴 x0=−b2ax_0 = -\frac{b}{2a}x0​=−2ab​

fmin=f(x0)=a(−b2a)2+b(−b2a)+cf_{\text{min}} = f(x_0) = a\left(-\frac{b}{2a}\right)^2 + b\left(-\frac{b}{2a}\right) + c fmin​=f(x0​)=a(−2ab​)2+b(−2ab​)+c

最大值：

fmax=max⁡{f(m),f(n)}f_{\text{max}} = \max\{f(m), f(n)\} fmax​=max{f(m),f(n)}

情况 2: x0<mx_0 < mx0​<m

函数在 [m,n][m, n][m,n] 上单调递增：

  • 最小值：f(m)f(m)f(m)
  • 最大值：f(n)f(n)f(n)

情况 3: x0>nx_0 > nx0​>n

函数在 [m,n][m, n][m,n] 上单调递减：

  • 最小值：f(n)f(n)f(n)
  • 最大值：f(m)f(m)f(m)
