# 数学试题记录站点

欢迎来到数学试题记录站点！这里是我记录和整理数学试题的地方。

## 站点特点
该站点是使用 VuePress 2.0.0-rc.26，和 @mdit/plugin-katex（数学公式渲染组件）构建的一个用于方便记录和整理数学试题的基础版本。以此为基础，可以方便的用于数学试题的记录和整理。克隆该站点，开启自己的数学试题记录站点。

[VuePress 站点制作可参考](https://mp.weixin.qq.com/s/wKkbSON0vDYy1qAGKPi2Tw)

自动部署到 Github Pages 参考 .github/workflows/deploy.yml

## 站点功能

- 📚 数学试题收集与整理
- 🧮 支持数学公式渲染
- 🔍 试题分类与索引
- 💡 详细解答过程

## 使用说明

1. 在"我的试题"中查看试题清单
2. 点击试题标题查看详细内容
